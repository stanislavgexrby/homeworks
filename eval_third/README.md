# Задание 3: Численное дифференцирование

## Файлы

1. **Вывод_формул_12_и_13.docx** - математический вывод формул (12) и (13) методом неопределенных коэффициентов (Задание 3.1)

2. **main.cpp** - программа численного дифференцирования (Задание 3.2)

## Компиляция и запуск программы

```bash
g++ -std=c++17 -o numerical_diff main.cpp
./numerical_diff
```

## Описание программы

Программа реализует:
- 4 тестовые функции (многочлены степени 2 и 3, быстро растущая, гладкая)
- Вычисление первой производной с точностью O(h²) (формулы 3, 4, 5)
- Вычисление первой производной с точностью O(h⁴) (формулы 7-11)
- Вычисление второй производной с точностью O(h²) (формулы 6, 12, 13)
- Сравнение с точными значениями производных

## Реализованные формулы

**Формула (12)** - вторая производная в начальной точке:
```
f''(x) = [2f(x) - 5f(x+h) + 4f(x+2h) - f(x+3h)] / h² + O(h²)
```

**Формула (13)** - вторая производная в конечной точке:
```
f''(x) = [2f(x) - 5f(x-h) + 4f(x-2h) - f(x-3h)] / h² + O(h²)
```
